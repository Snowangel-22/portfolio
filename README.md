# Portfolio v2

This is the source code for my personal portfolio. Built it from scratch to show how I think about product and how I build — same skills, applied to my own site.

**Live:** [snowangel-22.github.io/portfolio](https://snowangel-22.github.io/portfolio/)

---

Each project card opens a modal with the full breakdown — problem, what I built, and results. A few projects link out to standalone case study decks. There's a light/dark toggle in the navbar if that's your thing.

Stack: React 18, Vite, Tailwind CSS, Framer Motion. Deployed to GitHub Pages.

---

## Run it locally

```bash
npm install
npm run dev        # localhost:5173
npm run deploy     # build + push to gh-pages
```
